# Windrose Harbormaster

Is a HUD and Social companion for [Windrose](https://store.steampowered.com/app/3041230/). <br>
It adds a real-time player list overlay and lays the groundwork for more customization and modified HUD's within the game.

**No client-side UE4SS required** Players run a lightweight overlay `.exe`. Only the dedicated server needs UE4SS installed.

## Requirements

**Server:**
- Windrose Dedicated Server (Windows Server)
- UE4SS installed (`R5/Binaries/Win64/`) — use the [Windrose-tuned build](https://www.nexusmods.com/windrose/mods/43)
- Port `8766` open in your firewall / server panel

**Client (players):**
- Windows
- `WindroseHarbormaster.exe` + `WRHarbor_Client.ini` (provided by server owner)
- No UE4SS, no game files modified
